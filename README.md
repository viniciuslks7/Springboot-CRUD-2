# Springboot-CRUD-2

Este projeto é um CRUD Spring Boot com Thymeleaf, Tailwind CSS/DaisyUI, PostgreSQL e autenticação via Spring Security.

## Como rodar

1. Configure o banco de dados em `application.properties`.
2. Execute os scripts SQL em `db/migration`.
3. Rode `mvnw spring-boot:run`.

## Funcionalidades
- Cadastro e edição de oficinas
- Cadastro e edição de máquinas
- Autenticação de usuário
- Layout responsivo com DaisyUI

## Estrutura
- `src/main/java` - código Java
- `src/main/resources/templates` - templates Thymeleaf
- `src/main/resources/static/css` - CSS Tailwind
